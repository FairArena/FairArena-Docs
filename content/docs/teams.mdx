## What are Teams?

Teams are focused collaboration groups within organizations. They bring together people working on related projects and goals, providing structure, access control, and a dedicated workspace.

## Joining Teams

### Three Ways to Join

<Accordions>
  <Accordion title="1. Open Join (Instant)">
    Teams with open join enabled:
    
    **Steps:**
    1. Find the team
    2. Click **Join Team**
    3. Instantly become a member üéâ
    
    **Requirements:**
    - Team has "Open Join" enabled
    - You're in the organization
    - Meet any team requirements
  </Accordion>
  
  <Accordion title="2. Request Access">
    For closed teams you can view:
    
    **Steps:**
    1. Navigate to team page
    2. Click **Request Access**
    3. Write a message (optional but recommended)
    4. Submit request
    5. Wait for admin approval
    6. Get notification when approved
    
    **Tips for requests:**
    - Explain your interest
    - Mention relevant skills
    - Be professional
    - Follow up after a week if no response
  </Accordion>
  
  <Accordion title="3. Invitation Code">
    Secure invitation system:
    
    **Steps:**
    1. Receive invite code via email
    2. Click invitation link
    3. Review team details
    4. Accept invitation
    5. Select role (if options given)
    6. Join the team
    
    **About codes:**
    - Unique and secure
    - Role-specific
    - May have expiration
    - Single or multi-use
  </Accordion>

## Creating a Team

### Prerequisites

Before creating a team:

| Requirement | Description |
|-------------|-------------|
| **Organization Member** | Must be in an organization |
| **Permission** | Need team creation permission |
| **Purpose** | Clear goal for the team |
| **Structure** | Plan for roles and members |

### Creation Process

<Steps>

### Initiate Creation

Organization ‚Üí **Teams** ‚Üí **Create Team**

### Basic Information

**Team Name**
- Clear and descriptive
- Represents the team's purpose
- Example: "Engineering Team", "Marketing Squad"

**Team Slug**
- URL-friendly identifier
- Auto-generated from name
- Customizable
- Example: "engineering-team", "marketing"

**Description**
- Explain team's purpose
- What projects team handles
- Who should join
- Team goals and focus

### Set Visibility

Choose team visibility:

- ‚≠ï **Public** - Visible to everyone
- ‚≠ï **Internal** - Organization members only (recommended)
- ‚≠ï **Private** - Members only

### Join Settings

Configure access:

- ‚òê **Enable Open Join** - Anyone can join instantly
- ‚òê **Require Approval** - Manual approval for join requests

### Branding & Identity

**Team Logo**
- Recommended: 200x200px
- Formats: PNG, JPG, SVG
- Represents team visually

**Team Banner**
- Recommended: 1200x400px
- Appears on team page
- Creates visual impact

**Additional Details:**
- Location (optional)
- Website URL
- Timezone

### Create Roles

Define team roles:

| Role | Level | Typical Permissions |
|------|-------|---------------------|
| **Team Owner** | Highest | Full control |
| **Team Admin** | High | Manage most things |
| **Team Member** | Standard | Contribute to work |
| **Viewer** | Read-only | View content only |

### Invite Initial Members

Add your first team members:
1. Enter email addresses
2. Assign roles
3. Send invitations

### Launch Team

Click **Create Team** - your team is live! üöÄ

</Steps>

### Configure Code

| Setting | Options | Description |

**Do:**
- ‚úÖ Define clear team purpose

5. **Recognition:** Celebrate achievements

## Troubleshooting

<Accordions>
  <Accordion title="Can't create team">
    **Check:**
    - You're in an organization
    - You have permission to create teams
    - Organization has space for more teams
    - Name/slug not already taken
    
    **Solutions:**
    - Ask organization admin for permission
    - Try different name/slug
    - Check organization limits
  </Accordion>
  
  <Accordion title="Can't invite members">
    **Reasons:**
    - Insufficient permissions
    - Email already invited
    - User already in team
    - Organization limits reached
    
    **Solutions:**
    - Check your role permissions
    - Verify email address
    - Check if user is already member
    - Contact organization owner
  </Accordion>
  
  <Accordion title="Invite code not working">
    **Check:**
    - Code hasn't expired
    - Max uses not reached
    - Email matches (if specified)
    - User in organization (for Internal teams)
    
    **Solutions:**
    - Generate new code
    - Check code settings
    - Verify user requirements
  </Accordion>
  
  <Accordion title="Can't change team visibility">
    **Reasons:**
    - Insufficient permissions
    - Organization policy restrictions
    - Active projects prevent change
    
    **Solutions:**
    - Ask team owner/admin
    - Check organization settings
    - Review project visibility first
  </Accordion>
</Accordions>
