## Common Issues

<Accordions>
  <Accordion title="Can't log in">
    1. Check email/password spelling
    2. Clear browser cache
    3. Try different browser
    4. Use password reset
    5. Check 2FA code
  </Accordion>
  
  <Accordion title="Profile not showing">
    - Set profile to Public
    - Complete at least 50%
    - Verify email
    - Wait 24 hours for indexing
  </Accordion>
  
  <Accordion title="Payment failed">
    - Verify card details
    - Check balance
    - Try different payment method
    - Contact your bank
  </Accordion>
  
  <Accordion title="Invite code not working">
    - Check expiration
    - Verify not already used

    - Request new code
  </Accordion>
